WPFrank Companion is a companion plugin for WP Frank themes.

== Description ==

This is companion plugin is created for WP Frank themes. 
The plugin add custom frontpage sections for WP Frank themes and it will allow you full customizer settings for the theme front page sections. 
The plugin options are user-friendly and you can use this plugin for BusinessExpo WordPress theme.
